Variables in CSS

How to set variables in .css file

/* style.css */

:root,
:root.coffee {
  --bg-color: #b9b4ac;
  --text-color: #2B190C;
  --accent-color: #655e5a;
}
:root.blue {
  --bg-color: #343434;
  --text-color: #e1e7f0;
  --accent-color: #e4f4fc;
}

html {
	background-color: var(--bg-color);
	color: var(--text-color);
	border-color: var(--accent-color);
}

How to use the variables in .html files

<!-- file.html -->

<html lang="en" class="blue">
