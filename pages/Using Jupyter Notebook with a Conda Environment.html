Using Jupyter Notebook with a Conda Environment

Last updated: Sep. 29 2022

Terminal commands

Steps:

# Step 1: Create a new conda environment

conda create --name mynewenv

# Step 2: Activate the conda environment and modify it as you like

conda activate mynewenv

# (Modify with any installs, etc.)

pip install tqdm

# Step 3: CRITICAL STEP! This line allows jupyter notebook to use this environment

python -m ipykernel install --user --name=mynewenv

# Step 4: Deactivate the environment (this is not entirely necessary but it helps for remembering that when jupyter notebook is run, it doesn't run from the created environment automatically)

conda deactivate

# Step 5: Run jupyter notebook

jupyter notebook

Python code

How to check current environment in jupyter notebook

# Simply run the following code in a cell

import sys

print(sys.version, sys.platform, sys.executable)
