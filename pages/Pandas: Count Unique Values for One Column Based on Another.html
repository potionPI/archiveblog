Pandas: Count Unique Values for One Column Based on Another

Last updated: Nov. 2 2022

Code snippets

  • For each value in one column, count the number of unique values in another column

For each value in one column, count the number of unique values in another column:

# Find the animal type which has the most unique owner names by creating a dataframe which finds the count of unique owner names for each pet

import pandas as pd

df = pd.DataFrame({'Animal': [  'Falcon', 'Falcon', 'Falcon',
                                'Parrot', 'Parrot', 'Parrot'],
                    'Owner': ['Josh', 'Abraham', 'Minnie', 'Josh', 'Josh', 'Ron']})

grouped = df.groupby('Animal')['Owner'].nunique()

df:

    Animal	Owner
0	Falcon	Josh
1	Falcon	Abraham
2	Falcon	Minnie
3	Parrot	Josh
4	Parrot	Josh
5	Parrot	Ron

grouped:

Animal
Falcon    3
Parrot    2
Name: Owner, dtype: int64
