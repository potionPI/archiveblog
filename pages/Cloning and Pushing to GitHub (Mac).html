Cloning and Pushing to GitHub (Mac)

Last updated: Sep. 29 2022

How to clone a repository on a Mac terminal

# Navigate to the folder you're interested in saving the folder to first

cd /My/Folder/Path

# To copy the url associated with the repo, navigate to the repository online then click the "Code" button and copy the link under "HTTPS"

git clone https://github.com/username/reponame.git

How to push changes to a repository on a Mac terminal

# After making some changes to the folder obtained from cloning, in the same directory enter these commands in this terminal...

git add .

git commit -m 'my message on what channged'

git push

There may be a prompt for a username and "password" during this step. As of the time of writing, the "password" expected is actually a "personal access token." This token can be obtained from signing into GitHub and going into settings > developer settings > personal access tokens.
