Common import statement

import matplotlib.pyplot as plt

Basic plot and code structure

plt.figure()

x = [1, 2, 3]

y = [11, 8, 22]

plt.plot(x,y)

plt.show()

How to set title and axes labels

plt.title("My title")

plt.ylabel("y axis label")

plt.xlabel("x axis label")

How to change the size of a figure

width=15

height=5

plt.figure(figsize=(width, height))

Basic plot given all y values and their corresponding x values

x = [1, 2, 3]

y = [11, 8, 22]

plt.plot(x, y)

How to set range of the x and y axes using plt

x = [1, 2, 3]

y = [11, 8, 22]

x_lower = 0

x_upper = 5

y_lower = 5

y_upper = 25

plt.plot(x, y)

# Note that the range is INCLUSIVE! So for example, in the x axis, the number 0 will show up on lower left corner, and the number 5 will show up on the lower right corner.

plt.axis([x_lower, x_upper, y_lower, y_upper])

How to arbitrarily set location of multiple plots using fig.add_axes

Note: add_axes does NOT set the range of the axes, but rather the location and height and widths

fig = plt.figure()

# left plot

x = [1, 2, 3]
y = [11, 8, 22]
x0 = 0.1
y0 = 0.1
width = 0.5
height = 0.3
ax = fig.add_axes([x0, y0, width, height])
ax.plot(x,y)

# right plot

x = [1, 2, 3]
y = [12, 15, 2]
x0 = 0.5
y0 = 0.6
width = 0.8
height = 0.5
ax = fig.add_axes([x0, y0, width, height])
ax.plot(x,y)

plt.show()

How to plot multiple xy pairs on one plot (and add a legend)

The bulkier method which saves the plots in variables:

fig = plt.figure()

ax = fig.add_axes([1,1,1,1])

x = [1, 2, 3]
y = [11, 8, 22]
ax1 = ax.plot(x,y)

# Note that the basic line plot will NOT auto sort the x axis

x = [1, 5, 3]
y = [12, 15, 2]
ax2 = ax.plot(x,y)

ax.legend(['ax1 line', 'ax2 line'])

plt.show()

The lighter method which does NOT save the plots in variables:

fig = plt.figure()

x = [1, 2, 3]
y = [11, 8, 22]
plt.plot(x,y)

# Note that the basic line plot will NOT auto sort the x axis

x = [1, 5, 3]
y = [12, 15, 2]
plt.plot(x,y)

plt.legend(['ax1 line', 'ax2 line'])

plt.show()

How to zoom into (or out of) a section of the plot by setting limits on the axes

ax.plot(x,y)

# Note that the range is INCLUSIVE! So the lowest number on the x axis will be 1 and the highest number on the x axis will be 2. Likewise, both 8 and 12 will appear on the y axis.

ax.set_xlim(1,2)

ax.set_ylim(8,12)

How to make subplots

Lightweight method:

plt.figure()


# plt.subplot(total rows, total columns, current plot location)

plt.subplot(1,2,1)

x = [1, 2, 3]

y = [11, 8, 22]

plt.plot(x, y)


x = [1, 3, 4]

y = [12, 15, 2]

plt.subplot(1,2,2)

plt.plot(x, y)


plt.show()

Method that allows more flexibility:

One row or column:

# plt.subplots(total rows, total columns)

fig, axes = plt.subplots(2, 1)
fig.set_size_inches(w=10, h=5)

x = [1, 2, 3]
y = [11, 8, 22]
axes[0].plot(x,y)
x = [1, 3, 4]
y = [12, 15, 2]
axes[0].plot(x,y)
 
axes[1].plot(x,y)

More than one row/column:

# plt.subplots(total rows, total columns)

fig, axes = plt.subplots(2, 3)
fig.set_size_inches(w=10, h=5)

x = [1, 2, 3]
y = [11, 8, 22]

# axes[row_location, column_location]

axes[0,0].plot(x,y)

x = [1, 3, 4]
y = [12, 15, 2]

# axes[row_location, column_location]

axes[0,0].plot(x,y)

# axes[row_location, column_location]

axes[0,1].plot(x,y)

Another method that allows more flexibility, using a more flexible grid:

# plt.subplot2grid(shape_of_grid, location_of_top_left_corner, rowspan=1, colspan=1)

ax1 = plt.subplot2grid (
  (5, 3), (0, 0), rowspan = 2,  colspan = 2)
ax2 = plt.subplot2grid (
  (5, 3), (0, 2), rowspan = 5, colspan = 1)
ax3 = plt.subplot2grid (
  (5, 3), (2, 0), rowspan = 3, colspan = 2)

ax1.set_title('ax1')
ax2.set_title('ax2')
ax3.set_title('ax3')

x = [1,2,3]
y = [11,8,22]
z = [2,4,6]
ax1.plot(x, y)
ax2.plot(x, z)
ax3.plot(z, y)

# tight_layout() resolves title formatting issues with subplot2grid

plt.tight_layout()
plt.show()

How to control legend positioning

# With bbox_to_anchor, loc determines which corner of the legend box gets the coordinates specified by [x_position, y_position]

x_position=1.1

y_position=0.8

plt.legend(['ax1 line', 'ax2 line'], bbox_to_anchor=[x_position, y_position], loc='upper left')

# Without bbox_to_anchor, loc will position the legend based on the plot box itself.

plt.legend(['ax1 line', 'ax2 line'], loc='upper left')

Note: possible loc arguments include:

best
upper right
upper left
lower left
lower right
right
center left
center right
lower center
upper center
center

How to make a bar plot

# x axis entries need to be unique but need not be in order

# y axis is the height of the bar

x = [1, 5, 3]

y = [2, 1, 5]

plt.bar(x,y)

How to make a hist plot

# the plot will group items in x into bins

x = [1, 5, 3, 5]

plt.hist(x)

How to make a scatter plot

# x axis entries need not be unique nor in order

x = [1, 5, 3, 5]

y = [2, 1, 5, 3]

plt.scatter(x,y)

How to make a pie chart

# Each x value is its own pie slice. The percentage of the pie is automatically calculated by the plot where the total is the sum of all the x values

x = [400, 200, 300, 100]

# The greater the value, the more the pie slice corresponding to the position in the x array is "exploded" out from the pie chart

e =(0.1, 0, 0, 0.9)

plt.pie(x, explode = e)
