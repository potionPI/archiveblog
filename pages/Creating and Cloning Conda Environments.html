Creating and Cloning Conda Environments

Last updated: Sep. 9 2022

Terminal commands

How to create a new conda environment

# basic creation

conda create --name myenv

# specify python version

conda create -n myenv python=3.8

# don't ask for confirmation

conda create -n myenv -y

For more information, visit: >Anaconda reference for conda create command<

How to clone a conda environment

conda create --name myclone --clone myenv

How to switch in and out of a conda environment

# activate environment

conda activate myenv

# deactivate environment, a.k.a return to base environment

conda deactivate

How to get list of environments you have

conda env list
